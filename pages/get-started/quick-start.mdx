# Quick Start

First start by creating a JavaScript or TypeScript frontend.

When finished now install ankr.js

<Code terminal>npx @ankr.com/ankr.js --version</Code>

Once packages are installed `cd` into the directory of your project. Now in the same terminal start your Front End:

<Code terminal>dev</Code>
