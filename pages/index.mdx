### Version

Current Version [![Version](https://img.shields.io/npm/v/@ankr.com/ankr.js)](https://www.npmjs.com/package/@ankr.com/ankr.js)

When using `npx create-web3` you are saving this package locally on your machine. To check the current version on your machine

<Code>npx @ankr.com/ankr.js --version</Code>

To install the latest version

<Code>npm i @ankr.com/ankr.js</Code>

### Overview

⚓️ Ankr.js is a compact JavaScript library for interacting with Ankr's Advanced APIs.

### Issues

If you find a bug, an error in the docs, or have a suggestion, please go to [ISSUES](https://github.com/jcstein/ankrdocs/issues/new/choose) and open a new issue.

### Pull Requests

If you would like to fix a bug or error, please fork this repository, make your edits, and go to [Pull Requests](https://github.com/jcstein/ankrdocs/compare) and open a new PR.

### Discussions

If you have questions how to use, want to suggest a feature, or show off a project you created with ankr, join [discussions on GitHub](https://github.com/jcstein/ankrdocs/discussions). We would love to hear from you. 🙂
