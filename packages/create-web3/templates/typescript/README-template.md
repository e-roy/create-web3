## Quick Start Notes

To start the dev environment for hardhat:

```shell
yarn chain
```

In a 2nd terminal, for deploying contracts:

```shell
yarn deploy
```

In a 3rd terminal to start the dev server on your network:

```shell
yarn dev
```
